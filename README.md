# Python Pre-Mock Mini-Projects

A collection of three menu-driven Python applications built using a modular package architecture. Developed for the Pre-Mock Evaluation.

## 📂 Project Structure 

This repository contains three independent projects, separating the core logic from the main execution loop:

* **Project 1: ATM Simulation** - Handles deposits, withdrawals, and generates mini-statements using a transaction ledger.
* **Project 2: Stone-Paper-Scissors** - A randomized game logic engine evaluating user vs. computer choices.
* **Project 3: Library Management System** - A dictionary-based catalog system calculating progressive late fees based on allotted return dates.

## 🚀 Key Features
* Infinite `while True` menu loops for continuous user interaction.
* Exception handling to prevent crashes from invalid user inputs (e.g., entering text instead of numbers).
* Strict Package Architecture (`main.py` handles UI, `logic` handles logic).

## 🛠️ Tech Stack
* Python 3.x
* Built-in modules used: `random`, `datetime`, `math`
